# Trading Analyst — Course Starter

A stock analysis app you'll build step-by-step with Claude Code.

## Prerequisites

- Node.js 18+
- Docker Desktop
- Claude Code (`npm install -g @anthropic-ai/claude-code`)

## Quick Start

1. Start the backend:
   ```bash
   ./scripts/dc.sh up -d
   ```

2. Start the frontend:
   ```bash
   cd frontend
   npm install
   npm run dev
   ```

3. Begin the course:
   ```
   /introduce
   ```

## What You'll Build

A professional stock analysis tool with candlestick charts, technical indicators,
real market data, and a stock screening system — all built interactively with Claude Code.
