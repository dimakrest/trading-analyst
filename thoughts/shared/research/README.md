# Research Documents

This directory contains comprehensive codebase research documents created by the `/research_codebase` command.

## Structure
- Filename: `YYYY-MM-DD-topic-description.md`
- Contains: Detailed findings with file:line references
- Purpose: Document current state of codebase for future reference

## Usage
```bash
/research_codebase
# Answer: "How does setup validation work?"
# Creates: thoughts/shared/research/2025-01-15-setup-validation.md
```

## Document Format
Research documents include:
- Research question
- Summary of findings
- Detailed findings by component
- Code references with file:line numbers
- Architecture documentation
- Historical context (if relevant)
- Related research links
- Open questions

These documents serve as technical documentation of the codebase as it exists, helping developers understand implementation details, patterns, and architectural decisions.
